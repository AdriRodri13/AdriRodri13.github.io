Proyecto CRUD Java EE

Simulación de una lista de productos con funcionalidades CRUD: Crear, Leer, Actualizar y Eliminar.

Detalles del Proyecto CRUD en Java EE

Explora cada una de las funcionalidades CRUD: Crear, Leer, Actualizar y Eliminar, junto a un resumen final del proyecto.

Crear Producto

La funcionalidad de "Crear" permite añadir un nuevo producto a la base de datos. Se captura la información del producto a través de un formulario y se guarda en la base de datos usando Servlets y JSP.

Características: Captura de errores de entrada de datos.

Leer Productos

La funcionalidad "Leer" permite visualizar una lista de productos almacenados en la base de datos. Los datos se recuperan y se muestran en una tabla dinámica generada mediante JSP.

Características: Fácil Lectura de Datos.

Actualizar Producto

Con la funcionalidad de "Actualizar", los productos existentes pueden modificarse. Se puede seleccionar un producto para editarlo y luego guardar los cambios en la base de datos.

Características: Actualización intuitiva.

Eliminar Producto

La funcionalidad "Eliminar" permite borrar un producto específico de la base de datos. Esta acción es irreversible y se confirma antes de eliminar el producto.

Características: Rápido funcionamiento y actualización de la BBDD.

Resumen del Proyecto

Este proyecto CRUD implementado en Java EE permite una gestión completa de productos. Se desarrollaron todas las funcionalidades necesarias para crear, leer, actualizar y eliminar registros en la base de datos, siguiendo buenas prácticas de desarrollo con Java EE.

Tecnologías principales: Java EE, JSP, Servlets, MySQL, Bootstrap.
