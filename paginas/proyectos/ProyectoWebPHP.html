Proyecto Web con PHP

Proyecto de aplicacion web completamente funcional con PHP. Utilizacion de sesiones, consultas a BBDD, utilizacion de Modelo Vista Controlador. Una web completamente funcional.

Detalles del Proyecto Web

Aplicacion web para un laboratorio clinico, desarrollado para facilitar la creacion de contenido dentro de la web como la modificacion de la misma sin tener que tocar codigo Una aplicacion diseñada para ser util e intuitiva

Inicio

El usuario verá una pagina agradable, responsiva y con ciertas animaciones para darle dinamismo. Lo importante se esconde detrás, el Back-end de la pagina

Características: Agradable vista de usuario

El backend

Lo interesente de todo es El Backend usando el patron MVC he conseguido crear una web completamente editable para alguien que desconozca la programacion creando un producto realmente facil de utilizar. Implementando un CRUD en todos los casos

Características: Fácil de Usar

La programacionn

He implementado PDO y MVC para La programacion, del sitios web. Todo hecho con php 8.3, con el objetivo de usar un lenguaje sencillo, escalable y mantenible en el tiempo

Características: Mantenible en el tiempo.

BBDD

La BBDD "MySql" Nos permite almacenar tanto usuarios como productos, manejar diferentes aspectos de la web de forma rapida y segura, es la BASE del CRUD.

Características: Rápido funcionamiento y actualización de la BBDD.

Resumen del Proyecto

Este proyecto web con PHP es la puesta en practica de mis conocimientos de programador. Con el objetivo de crear una web completamente funcional tanto para el usuario que navegue por ella como para el administrador de la misma, he creado una web agradable y funcional en constante crecimiento

Tecnologías principales: PHP, MySQL, JS, HTML Y CSS.
